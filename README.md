# 概率论与数理统计
## 笔试题相关
携程2020届算法工程师笔试题(**当时没有做出来**):
有一个木桶，里面有M个白球，小明每分钟从桶中随机取出一个球涂成红色（无论白或红都涂红）再放回，问小明将桶中球全部涂红的期望时间是多少？

**解题思路:**  设E(i)代表M个球中已经有i个球是红色后，还需要让剩下所有的球变成红色球的期望。
